🌌 Solar System Animation
📌 About:-
This project is a Solar System animation created using HTML and CSS.
It shows the Sun at the center and planets revolving around it.
The Earth also has a Moon orbiting around it.

🚀 Features:-
☀️ Sun at the center
🪐 All planets orbiting around the Sun
🌍 Earth with Moon animation
🎨 Smooth CSS animations
🛠️ Technologies Used
HTML5 – for structure
CSS3 – for design and animation
📂 Files
index.html – main structure
style.css – styling and animation
script.js – (optional)
▶️ How to Run
Download the project
Open the folder
Double click on index.html
💡 Working
Each planet is inside an orbit (circle)
CSS @keyframes is used for movement
transform: rotate() + translateX() creates circular rotation
Moon is inside Earth, so it moves around it

🎯 Purpose:-
This project is made to practice:

CSS animations
Positioning (absolute & transform)
Creating real-world animations

✨ Output:-
An animated solar system where all planets move continuously.
